Orb configuration

How many orbs, and should the component be configurable?

A

Fixed: 1 large + 2 small

Matches the reference exactly — one dominant orb in the center, two small accent orbs. Hardcoded layout.

B

Configurable via props

Pass orb count, sizes, and color palette as props. Default matches the reference, but you can customize per screen. e.g. <OrbBackground orbs={[{size: 'lg'}, {size: 'sm'}, {size: 'sm'}]} />

Color palette

Match the reference — deep purple core with blue/pink/cyan reflections and white specular highlights.

D

Theme-adaptive palette

Purple iridescent for dark mode, softer blue-lavender for light mode. Adapts to your existing glass theme system.
