How should the orbs be rendered?

Trade-off: visual fidelity vs. complexity and performance. Your project currently has Reanimated, SVG, LinearGradient, and Blur — but no Skia or GL.

A

Animated gradient blobs

Radial gradients with blur and glow effects using existing libs (LinearGradient + BlurView + Reanimated). Soft, glowy orbs — atmospheric but not photorealistic glass. No new deps.

B

Skia shader orbs

Add @shopify/react-native-skia — write GLSL-like shaders for true iridescent glass with refraction, specular highlights, and organic morphing. Closest to the reference image. New dependency, native rebuild required.

C

Pre-rendered orb assets

Use high-quality PNG/Lottie orb assets with Reanimated for movement and scale. Looks exactly like the reference because it is the reference aesthetic. No new deps, but less dynamic morphing.

D

WebGL via expo-gl

Use expo-gl with raw WebGL or Three.js for full 3D glass sphere rendering with environment mapping. Maximum fidelity. New dependency, heavier on GPU, native rebuild.
