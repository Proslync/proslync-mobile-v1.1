Design: Interactive Floating Orb Background

Standalone <OrbBackground /> component — full interactive glass orbs via WebGL

Architecture

│ │ │

How the Glass Effect Works

Each orb is a Three.js sphere with MeshPhysicalMaterial:

  • transmission: 0.95 — near-fully transparent glass
  • roughness: 0.05 — mirror-smooth surface
  • ior: 2.4 — high refraction index (diamond-like)
  • iridescence: 1.0 — rainbow color shifting at angles
  • iridescenceIOR: 1.5 — controls iridescence spread
  • specularIntensity: 1.0 — bright white highlights
  • envMap — generated cubemap with purple/blue/pink gradient

Organic morphing via vertex displacement using sine waves over time.

Physics & Interaction

  • Ambient drift: Orbs float with Perlin noise-based paths
  • Drag: Pan gesture raycasts into scene, moves nearest orb
  • Collision: Simple sphere-sphere overlap detection with spring pushback
  • Edge bounce: Orbs spring back when hitting screen bounds
  • Gyroscope: Device tilt adds directional force to all orbs
  • Tap wobble: Nearest orb gets a vertex noise spike that decays

Performance

  • GL renders on a separate thread — won't block RN JS thread
  • Low poly spheres (32 segments) — sufficient for glass illusion
  • Auto-pauses render loop when component unmounts or app backgrounds
  • Gyroscope updates throttled to 30Hz
